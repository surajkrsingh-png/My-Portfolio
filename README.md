# Personal Portfolio Website

A personal portfolio website built using HTML and CSS to showcase my skills, projects, and learning journey as a web developer.

## Features

- Responsive portfolio layout
- About Me section
- Skills section
- Projects showcase
- Contact section
- Modern and clean user interface
- Smooth styling and animations

## Technologies Used

- HTML5
- CSS3

## Project Purpose

This project was created to:

- Practice front-end web development
- Improve HTML and CSS skills
- Build a professional online presence
- Showcase projects and achievements

## Sections Included

- Home
- Skills
- Testimony
- Contact

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Learning Outcomes

Through this project, I learned:

- Semantic HTML structure
- CSS Flexbox and Grid
- Responsive design principles
- CSS animations and transitions
- Portfolio website design
